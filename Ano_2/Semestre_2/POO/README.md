# 🎵 SpotifUM

**SpotifUM** is a music management ecosystem developed in **Java**, inspired by the Spotify experience. Created in the context of the Object-Oriented Programming (OOP) course, this project uses a command-line interface (CLI) to simulate complex interaction flows between users and multimedia content.

## 🛠️ Architecture and Features

The application was designed focusing on the principles of **encapsulation, inheritance, and polymorphism**:

* **Identity Management:** Authentication system with distinction between regular users and administrators.
* **Music Ecosystem:** Organization of tracks, albums, and playlists, including automatic generation algorithms based on preferences.
* **Business Models:** Implementation of subscription plans that unlock premium features.
* **Data Persistence:** Save/Load mechanism to maintain application state between sessions.
* **Quality Assurance:** Unit test suite with **JUnit** and technical documentation via **Javadoc**.

The project uses **Gradle** to simplify lifecycle management:

| **Run Tests** | `./gradlew test` |
