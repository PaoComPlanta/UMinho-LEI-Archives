# 🌐 Computer Networks | UMinho

Repository dedicated to archiving practical assignments and network infrastructure projects developed in the Computer Networks course at the University of Minho.

## 📋 About the Course

This course focuses on the fundamental principles and practical implementation of computer networks. Throughout the semester, we explored network architectures, protocols, and communication systems, covering:

* **Network Protocols:** Analysis and implementation of TCP/IP stack protocols.
* **Network Architecture:** Design and configuration of network topologies and routing.
* **Network Analysis:** Packet capture and traffic analysis using Wireshark.
* **Network Simulation:** Virtual network environments and performance evaluation.

## 🛠️ Technologies and Tools

* **Network Simulation:** CORE/IMUNES
* **Traffic Analysis:** Wireshark/tcpdump
* **Protocols:** TCP/IP, UDP, HTTP, DNS
* **Documentation:** Network diagrams and technical reports

## 📂 Project Structure

The repository is organized by practical assignments:

* **TP0:** Introductory exercises
* **TP1:** Network fundamentals and protocol analysis
* **TP2:** Network topology design and implementation
* **TP3:** Advanced routing and network configuration

Each assignment folder contains:
* Assignment specifications (PDF)
* Network configuration files
* Packet capture files (.pcapng)
* Technical documentation and reports
