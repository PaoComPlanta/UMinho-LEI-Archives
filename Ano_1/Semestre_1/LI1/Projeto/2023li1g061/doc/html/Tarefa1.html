colisoesParede #

Arguments

:: Mapa

Mapa do Jogo.

-> Personagem

Personagem em causa.

-> Bool

Valor lógico da colisão entre os limites do mapa / plataformas e o personagem.

A função colisoesParede testa se uma personagem se encontra em colisão com algum dos limites do mapa (topo ou laterais), ou com algum bloco de plataforma. Para nos auxiliar nesta tarefa utilizamos algumas funções que realizam tarefas mais específicas, como por exemplo :

  • colisaoMapa: Testa se uma personagem colide com algum dos limites do mapa
  • colisaoBlocos: Testa se uma personagem colide com um bloco de plataforma
  • hitboxPersonagem: Calcula a hitbox de uma personagem, definifida por um par de posições, sendo estas o canto inferior esquerdo e superior direito
  • hitboxBlocos: Calcula a hitbox de uma lista de plataformas
  • posPlataforma: Recebe a matriz de definição do mapa e devolve uma lista com as coordenadas do canto superior esquerdo de cada bloco

Propriedades:

Função principal "colisoesParede":

colisoesParede :: Mapa -> Personagem -> Bool
colisoesParede (Mapa _ _ m) p
| colisoesParede m p || colisaoBlocos (hitboxBlocos $ posPlataforma m) (hitboxPersonagem p) = True
| otherwise = False

Função auxiliar "colisaoMapa":

colisaoMapa :: [[Bloco]] -> Personagem -> Bool
colisaoMapa m (Personagem _ _ (x, y) _ (g, c) _ _ _ _ _)
| x - (g / 2) <= 0 || x + (g / 2) >= th || y - (c / 2) <= 0 || y + (c / 2) >= tv = True
| otherwise = False
where
th = fromIntegral $ length $ head m
tv = fromIntegral $ length m

Função auxiliar "colisaoBlocos":

colisaoBlocos:: [Hitbox] -> Hitbox -> Bool
colisaoBlocos [] _ = False
colisaoBlocos (((xi1, yi1), (xi2, yi2)):xs) ((xe1, yd1), (xe2, yd2)) =
elem True [x >= min xe1 xe2 && x <= max xe1 xe2 && y >= min yd1 yd2 && y <= max yd1 yd2 | (x, y) <- v] || colisaoBlocos xs ((xe1, yd1), (xe2, yd2))
where
v = [(x, y) | x <- [xi1, xi2], y <- [yi1, yi2]]

Função auxiliar "posPlataforma":

posPlataforma :: [[Bloco]] -> [Posicao]
posPlataforma [] = []
posPlataforma m =
concatMap (\(l, bs) -> [(fromIntegral c, fromIntegral l) | (b, c) <- zip bs [0..], b == Plataforma]) $ zip [0..] m

Função auxiliar "hitboxBlocos":

hitboxBlocos :: [Posicao] -> [Hitbox]
hitboxBlocos = map (\(x, y) -> ((x, y + 1), (x + 1, y)))

Função auxiliar "hitboxPersonagem":

hitboxPersonagem :: Personagem -> Hitbox
hitboxPersonagem (Personagem _ _ (x,y) _ (g,c) _ _ _ _ _)
= ((xe1,yd1),(xe2,yd2))
where
xe1 = x - (g / 2)
yd1 = y + (c / 2)
xe2 = x + (g / 2)
yd2 = y - (c / 2)

>>> colisoesParedes (Mapa ((0, 0), Este) (4, 4) [[Plataforma, Plataforma, Plataforma], [Plataforma, Vazio, Plataforma], [Plataforma, Plataforma, Plataforma]]) (Personagem (2, 0) Jogador (1, 1) Este (1, 1) False False 5 0 (False, 0))

>>> colisoesParedes (Mapa ((0, 0), Este) (4, 3) [[Vazio, Vazio, Vazio], [Vazio, Vazio, Vazio], [Vazio, Vazio, Vazio]]) (Personagem (2, 0) Jogador (1, 1) Este (1, 1) False False 5 0 (False, 0))
False

Valor lógico da colisão entre a Personagem 1 e a Personagem 2.

A função colisoesPersonagens testa se dois personagens se encontram em colisão, ou seja, testa se a hitbox de um dos personagens interceta de alguma forma a hitbox do outro Para definir esta função, utilizamos a função hitboxPersonagem a também uma outra função auxiliar :

  • colisaoHitbox: Verifica se duas hitboxes se intercetam

colisaoHitbox :: Hitbox -> Hitbox -> Bool
colisaoHitbox ((xi1, yi1), (xi2, yi2)) ((xe1, yd1), (xe2, yd2)) =

where
v = [(x, y) | x <- [xi1, xi2], y <- [yi1, yi2]]
