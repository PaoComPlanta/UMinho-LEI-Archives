nextVogal :: Char -> Char #

A função nextVogal recebe um caractere e retorna a próxima vogal após esse caractere, considerando que a próxima vogal após u é a. Caso o caractere não seja uma vogal (aeiou), a função mantém o caractere original.

Exemplo de Uso

>>> nextVogal 'e'
'i'
>>> nextVogal 'i'
'o'

Propriedades

nextVogal 'a' = 'e'
nextVogal 'e' = 'i'

|c == 'a' = 'e' : nvogal cs
| c == 'e' = 'i' : nvogal cs
| c == 'i' = 'o' : nvogal cs
| c == 'o' = 'u' : nvogal cs
| c == 'u' = 'a' : nvogal cs

posicao #

Arguments

:: PosicaoPessoa

Posição inicial.

-> Movimentos

Sentido dos movimentos

-> PosicaoPessoa

Posição após ocorrer movimento

A função posicao calcula a posição de uma pessoa depois de executar uma sequência de movimentos:

posicao :: PosicaoPessoa
             -> Movimentos
             -> PosicaoPessoa
posicao (Pos n (x,y)) [] = Pos n (x,y)
posicao (Pos n (x,y)) (m:ms) =
    case m of
        N -> posicao (Pos n (x,y+1)) ms
        S -> posicao (Pos n (x, y-1)) ms
        E -> posicao (Pos n (x+1,y)) ms 
        W -> posicao (Pos n (x-1,y)) ms

Exemplos de utilização:

>>> posicao (Pos "Josuke" (1,2)) [N,E]

A função posicoesM é uma função que dada uma lista de posições de pessoas, actualiza essa lista depois de todas executar um movimento dado:

Exemplo de utilização:

>>> posicoesM [Pos "Joel" (0,0),Pos "Ellie" (3,2),Pos "Abby" (2,6)] N
[Pos "Joel" (0,1),Pos "Ellie" (3,3),Pos "Abby" (2,7)]

Propriedades:

posicoesM [] _ = []
posicoesM (Pos n (x, y):rn) m =
case m of
N -> Pos n (x, y + 1) : posicoesM rn m -- Norte (aumenta y)
S -> Pos n (x, y - 1) : posicoesM rn m -- Sul (diminui y)
E -> Pos n (x + 1, y) : posicoesM rn m -- Leste (aumenta x)
W -> Pos n (x - 1, y) : posicoesM rn m -- Oeste (diminui x)

A função posicoesMs é uma função que dada uma lista de posições de pessoas, actualiza essa lista depois de todas as pessoas executarem uma mesma sequência de movimentos. Esta função usa as funções anteriormente definidas.

Exemplo de utilização:

>>> posicoesMs [Pos "Joel" (0,0),Pos "Ellie" (3,2),Pos "Abby" (2,6)] [N,E]

Propriedades:

posicoesMs [] _ = []
posicoesMs (p:ps) m =
let pa = posicao p m
in  pa : posicoesMs ps m

Lista de pessoas e respetivas posições

Nome(s) de pessoa(s) mais a Norte

A função pessoasNorte é uma função que calcula quais são as pessoas posicionadas mais a norte.

Exemplo de utilização:

>>> pessoasNorte [Pos "Joel" (0,0),Pos "Ellie" (3,2),Pos "Abby" (2,6)]
"Abby"

Propriedades:

pessoasNorte [] = []
pessoasNorte posicoes = aux posicoes [] maxCoordY
where
aux :: [PosicaoPessoa] -> [Nome] -> Int -> [Nome]
aux [] nomes _ = nomes
aux ((Pos nome (x, y)):resto) nomes maxY
| y > maxY = aux resto [nome] y
| y == maxY = aux resto (nome : nomes) maxY
| otherwise = aux resto nomes maxY
maxCoordY :: Int
maxCoordY = maximum [y | (Pos _ (_, y)) <- posicoes]

deslocaEsquerda :: [a] -> [a] #

A função dlista é uma função que recebe uma lista e desloca cada elemento da lista, n posições para a direita.

Exemplo de utilização:

>>> dlista 2 [1,2,3,4,5]
[4,5,1,2,3]

Propriedades:

dlista _ [] = []
dlista n lista =
let tamanho = length lista
deslocamento = n `mod` tamanho
(inicio, fim) = splitAt (tamanho - deslocamento) lista
in fim ++ inicio
