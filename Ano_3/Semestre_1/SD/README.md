# 🌐 Sistemas Distribuídos

<div align="center">

**Universidade do Minho** | Licenciatura em Engenharia Informática

*Trabalho desenvolvido na unidade curricular de Sistemas Distribuídos*

</div>

---

## 📋 Visão Geral

A cadeira foca-se no **design e implementação de aplicações distribuídas**, abordando:

- 🔒 Controlo de concorrência
- 🔌 Programação com sockets
- 🔄 Primitivas de sincronização
- 🛡️ Tolerância a falhas

Inclui diversos exercícios práticos e um **projeto final** que envolve uma arquitetura cliente-servidor.

---

## 🛠️ Stack Tecnológica

| Categoria | Tecnologia |
|-----------|------------|
| **Linguagem** | Java ☕ |
| **Build Tool** | Gradle 🐘 |
| **Networking** | Sockets 🔌 |
| **Concorrência** | Multi-threading (Locks, Conditions) 🧵 |

---

## 📝 Nota

> O código base para vários exercícios e a estrutura do projeto foram fornecidos pela equipa docente. O trabalho desenvolvido consistiu na **implementação da lógica distribuída**, **mecanismos de sincronização** e **protocolos de rede**.
