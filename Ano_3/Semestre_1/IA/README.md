# 🧠 Inteligência Artificial

Esta pasta contém o trabalho desenvolvido durante a unidade curricular de **Inteligência Artificial** na Universidade do Minho.

---

## 📖 Descrição

A disciplina aborda conceitos fundamentais de IA, incluindo:
- **Estratégias de Pesquisa:** Informada e Não-Informada
- **Algoritmos de Otimização**
- **Programação em Lógica**

O trabalho prático envolve a implementação de:
- Algoritmos de pesquisa: **A***, **BFS**, **DFS**
- Pesquisa local: **Hill Climbing**, **Simulated Annealing**
- Representação de conhecimento utilizando **Prolog**

---

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** Python 🐍 | Prolog 📜
* **Tópicos:** 
    - Pesquisa em Grafos
    - Heurísticas
    - Programação em Lógica
    - Satisfação de Restrições

---

## 🙏 Agradecimentos

A estrutura dos projetos e exercícios foi desenvolvida no âmbito da unidade curricular. Alguns módulos auxiliares e especificações de problemas foram fornecidos pelo corpo docente para orientar a implementação de modelos e algoritmos de IA.
