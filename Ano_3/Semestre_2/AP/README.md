## 📖 About

**AP (Análise de Projetos)** focuses on the financial and economic evaluation of investment projects. It covers the methodologies and techniques used to assess the viability, profitability, and risks associated with business ventures and capital investments.

This repository contains:
- 📈 Financial modeling and cash flow projections
- 💶 Investment appraisal calculations (NPV, IRR, Payback)
- 📊 Risk and sensitivity analysis
- 📑 Final project report and interactive spreadsheets

### 🎯 Learning Objectives

- Construct comprehensive financial models for investment projects
- Calculate and interpret key financial indicators (Net Present Value, Internal Rate of Return)
- Perform sensitivity analysis to assess project risks and uncertainties
- Understand cost-benefit analysis and economic viability
- Use spreadsheet software (Excel/VBA) for complex financial forecasting

## 🚀 Getting Started

### Prerequisites

To view and interact with the financial models in this repository, you will need:
- Microsoft Excel (or a compatible spreadsheet software that supports `.xlsm` files)

### Opening the Project

1. Clone or download the repository.
2. Navigate to the `AP` directory.
3. Open the `Grupo_2.xlsm` file in Microsoft Excel.
4. **Important**: Since the workbook contains macros, you may need to click **"Enable Content"** or **"Enable Macros"** in the security warning bar at the top of the Excel window to ensure all automated calculations and scripts function correctly.
5. Review the `Grupo_2.pdf` report for a detailed explanation of the methodology, assumptions, and conclusions of the analysis.
